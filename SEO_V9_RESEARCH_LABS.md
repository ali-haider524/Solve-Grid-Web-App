# V9 Research Labs SEO Plan

## Primary landing page

`/research-tools`

Intent: research calculation tools, numerical modelling tools, engineering simulation calculators.

## Differential equations cluster

Primary page: `/differential-equation-solver`

Visible capabilities:

- differential equation solver
- Euler method calculator
- Improved Euler / Heun method calculator
- RK4 / Runge–Kutta calculator
- numerical ODE solver
- phase plot calculator

Supporting guides:

- `/guides/euler-method-for-differential-equations`
- `/guides/runge-kutta-rk4-method`

## Content rule

Use the capabilities where they genuinely describe a working feature. Do not create thin route variations like `/rk4-calculator`, `/euler-calculator`, and `/ode-solver-online` if they all lead to the same identical interface. Build a separate page only when the calculation workflow is meaningfully different.

## Future research workspaces

Only add future routes when a functional workspace is ready:

- symbolic algebra
- optimization lab
- circuit simulation
- advanced statistics

Each requires its own verified calculation method, assumptions, guides, route registry entry, schema, sitemap entry, responsive QA, and internal links.
