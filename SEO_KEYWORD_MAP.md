## Content choices
- Use the primary intent in the page title, H1, first paragraph, and visible tool description naturally.
- Use supporting phrases only where they accurately describe a visible, working feature. For example, do not target “eigenvalue calculator” unless the page shows the functioning eigenvalue workflow.
- Avoid repeating a target phrase unnaturally in headings, alt text, or hidden content.
