# V10 navigation repair and research-labs expansion

## Navigation
- Desktop `Tools` now opens a compact category menu instead of a four-column list of every tool.
- Mobile uses a hamburger drawer with collapsed category panels.
- The header stays in document flow and is opaque, so it does not overlap calculator outputs.

## Research labs added
- Symbolic Algebra Lab: simplify, differentiate, evaluate, and compare expression forms.
- Optimization Lab: two-variable continuous linear-programming corner-point analysis.
- DC Circuit Analysis Lab: ideal Ohm, resistor-network, voltage-divider, RC, and two-node nodal calculations.
- Advanced Statistics Lab: one-sample t-test, Welch t-test, correlation/regression, and one-way ANOVA.

## Deliberate boundaries
- Symbolic comparison is not a theorem prover.
- Optimization is not an integer, binary, or nonlinear optimizer.
- Circuit Analysis is not a SPICE simulator.
- Statistics are decision-support outputs with assumptions, not research conclusions.
